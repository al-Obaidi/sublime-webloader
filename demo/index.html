Start editing demo.css or demo.less! Changes should be shown as you type.

You can also edit index.html, change something, then save. The page should refresh.

This plugin is still beta. If you see no changes, you can try the following:

  • - restart Sublime Text 2, open the console (ctrl-`), and look for any Webloader: messages
  • - open the javascript console (ctrl-shift-j in chrome), look for [webloader] warnings, or js errors
  • - find help on the project page

    • If you are using this on your pages, or want to dig around:

  • - make sure you included the webloader.js script on the page you want to refresh (like in the source of this page)
  • - check out the 'sites' setting in Sublime Text 2/Packages/Webloader/Webloader.sublime-settings
  • - find the Sublime Text 2/Packages/Webloader/webloader.log file, see if there are any errors

    • If you are willing to test and help:

  • - please narrow down the problem a bit before reporting, if possible
  • - submit it on the github issues page (or send via email, if necessary)
